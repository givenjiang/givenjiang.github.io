工作流程

那么，你希望为你的图片或者视频换脸是吗？好的，请稍等一下，因为首先你需要理解faceswap都在干什么，它是如何工作的以及它的缺陷。

faceswap的基本操作流程是简单的。它通过训练一个机器学习模型来识别和交换两个面部图像。这个机器学习模型可以看作是我们的小小“机器人”，我们可以使用图像作为训练数据调教它，从而使其真正具有“交换”的能力。需要注意的是，我们的模型主要用于处理人脸，对于其他的目标可能效果不好。

接下来是我们的计划，我们希望川普在总统竞选的时候输给尼古拉斯凯奇；我们有川普的就职演说视频；让我们开始把川普换脸成尼古拉斯凯奇吧。

获取原始数据

为了完成上述任务，模型需要学会去识别face A（川普）和face B（凯奇）。在一开始，模型并不知道川普或者凯奇的样子。所以我们需要向它展示一些图片并让它预测图片的类别。接下来我们首先来收集所需的面部。

通过Google, DuckDuckGo或者Bing图片搜索，我们可以获得大量的图片。或者，我们也可以将一些视频转换为图片。参考分解视频帧章节以了解更多。

也欢迎你在faceswap-playground上分享你的图片集或者为官方贡献你的想法。

现在，我们拥有了一系列川普和凯奇的图片。让我们将保存它们保存到faceswap工程的目录下。例如：~/faceswap/photo/trump 或者 ~/faceswap/photo/cage

提取面部数据

现在我们面临一个问题，尽管我们拥有成吨的图像数据，但是这些图像里既包含川普和凯奇，也包含了背景以及其他不相关的人。然而我们只能通过面部数据来训练模型，下面的命令可以实现对于面部数据的提取：

# To convert trump:
python faceswap.py extract -i ~/faceswap/photo/trump -o ~/faceswap/data/trump
# To convert cage:
python faceswap.py extract -i ~/faceswap/photo/cage -o ~/faceswap/data/cage

在上面的命令参数中，我们需要指定输入图像的路径和输出数据的路径，这些输出数据将会被当作训练数据存储。命令执行的时候，程序会识别面部的特征并且将其截取出来保存到输出路径。需要注意的是：这个面部提取模块并不完美，它可能会错误地提取到非目标的面部。所以：在开始训练一定要检查好训练集的数据。训练数据将会在很大程度上影响模型的表现。

你可以通过-h 标志查看extract命令所支持所有参数：

python faceswap.py extract -h

有一些插件支持自定义选项。你可以在以下文件中找到它们：<faceswap_folder>\plugins\extract\config.ini<faceswap_folder>\plugins\extract\config.ini.需要至少运行一次提取操作以生成这个配置文件。

模型训练

训练将会占据整个流程的绝大部分，特别是当你使用CPU时。在命令中，我们需要给出两个面部数据的路径，以及模型的存储路径。当这个命令运行之后就会开启训练流程。我个人倾向于不断地预览输出结果，一旦结果让我满意我就会停止训练。

python faceswap.py train -A ~/faceswap/data/trump -B ~/faceswap/data/cage -m ~/faceswap/models/
# or -p to show a preview
python faceswap.py train -A ~/faceswap/data/trump -B ~/faceswap/data/cage -m ~/faceswap/models/ -p

如果开启了预览功能，那么可以选中预览窗口并按下回车键以保存窗口中的图像并且退出。如果你没有开启预览功能，那么你需要通过Ctrl+C来强制中断命令的执行。需要注意的是，每当训练迭代次数达到100时都会自动保存模型，但是保存过程会花些时间。所以请在确保模型保存成功后再停止整个流程。

你可以通过-h 标记查询训练操作所有可能的参数：

python faceswap.py train -h

有一些插件支持自定义选项。你可以在以下文件中找到它们：<faceswap_folder>\plugins\traom\config.ini.需要至少运行一次提取操作以生成这个配置文件。

转换流程

现在我们已经获得了训练好的模型，可以开始转换视频了。与提取面部流程相似，转换流程会使用相同的程序检测目标面部，并且截取面部到正确的尺寸，然后在这张图片上进行换脸，最后将换脸后的面部图片粘贴回原图像中。

让我们开始将川普的脸换成凯奇的吧。首先我们需要在命令中指定川普照片的路径作为输入路径，指定一个新的文件夹作为保存输出文件的路径，最后需要指定采用预训练模型的路径：

python faceswap.py convert -i ~/faceswap/photo/trump/ -o ~/faceswap/output/ -m ~/faceswap/models/

你可以通过-h 标记查询转换操作所有可能的参数：

python faceswap.py convert -h

用户界面

以上的所有命令和选项都可以在用户界面中执行，执行以下命令可以打开用户界面：

python faceswap.py gui

视频序列

视频只是由一系列图片组成的连续帧序列，所以你可以将你的数据集图片或者转换好的图片合成为视频序列。

EFFMPEG

你可以通过内建的EFFMPEG工具对视频进行处理，effmpeg的命令参数可以通过以下命令查看：

python tools.py effmpeg -h

使用FFMPEG拆分视频序列

这里你可以使用ffmpeg来将视频拆分成独立的图像帧序列，下面是一个示例命令：

ffmpeg -i /path/to/my/video.mp4 /path/to/output/video-frame-%d.png

（将独立图像帧序列）合成视频

你也可以选择使用ffmpeg将已经换脸的帧序列合成为视频文件，下面的命令展示了如何将png格式的图片帧合成为一个视频文件：

ffmpeg -i video-frame-%0d.png -c:v libx264 -vf "fps=25,format=yuv420p" out.mp4

注意事项

这个教程远非完整，程序的功能可能还会有变化，新的依赖可能在未来也会有变化。

如果你遇到了问题，欢迎你将问题提交到faceswap-playground仓库而不是本仓库。
